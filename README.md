# ⚔️ TG-RPG v5.0.1

Фэнтези RPG для Telegram Mini App (и браузера).

---

## 📋 Changelog

### v5.0.1 (текущая)
**Исправления:**
- 🐛 `styles/main.css` был пустым — все экраны отображались одновременно как HTML без стилей. Написан полный CSS (~15KB).
- 🐛 `const sound = new Sound()` не был объявлен нигде — любой вызов `sound.play()` крашил игру (атака, подбор предметов, вход в подземелье).
- 🐛 `PORTAL_POS` был объявлен дважды (`Portal.js` + `config.js`) — `SyntaxError` при старте.
- 🐛 `showQNotif()` нигде не была объявлена, хотя вызывалась повсюду.
- 🐛 Рябь на экране деревни — `makeVillageCache()` с `Math.random()` вызывался каждый кадр. Добавлен кэш деревни.
- 🐛 Координаты `Item` в подземелье были в пикселях в одном месте и тайлах в другом — предметы не отображались. Унифицировано: tile-координаты хранятся, `draw()` конвертирует сам.
- 🐛 `NPC.talk()` никогда не вызывался — магазин и квесты были недоступны. Теперь кнопка 🌀 проверяет близость к NPC.

**Новое:**
- 🆕 Сплэш-экран на 3 секунды с прогресс-баром (поддерживает `assets/splash.png`).
- 🆕 Главное меню после сплэша — показывает карточку сохранения (имя, уровень, глубина), кнопки «Продолжить» / «Новая игра» / «Удалить сохранение».
- 🆕 Умный выбор: если сейва нет — сразу «Начать игру»; если есть — «Продолжить» становится главной кнопкой.

## 🚀 Установка

Разархивируй в папку и открой `index.html` в браузере или задеплой на GitHub Pages / Telegram Mini App.

## 🖼️ Своё изображение на сплэш

Положи файл `assets/splash.png` (рекомендуется 360×360 px) — он автоматически появится на загрузочном экране.
