# ⚔️ TG-RPG v5.3.1

Фэнтези RPG для Telegram Mini App (и браузера).

---

## 📋 Changelog

### v5.3.1 (текущая)
**Новое:**
- 🆕 **Line of Sight** (Брезенхэм по тайлам) — лучник и маг атакуют только врагов с прямой видимостью. Стрела и файрбол не летят сквозь стены подземелья. Мечник (ближний бой) стены не учитывает.
- 🆕 **A* Pathfinding** — враги обходят стены по оптимальному маршруту. Путь пересчитывается раз в ~30 тиков или при смене позиции игрока. При прямой видимости (LoS) враг идёт напрямую без A*. Поддержка диагонального движения. Лимит 300 узлов предотвращает зависание на сложных картах.
- 🆕 Враги теперь скользят вдоль стен (раздельная проверка осей X/Y) вместо полной остановки.

### v5.3.0 (предыдущая)
**Новое:**
- 🆕 **SpriteConfig.js** — централизованный конфиг анимаций в `localStorage`. Каждая анимация имеет свой файл, число кадров, FPS и звук. Поддерживает отдельные файлы на каждое действие (idle, walk, attack, ability, death).
- 🆕 **Гибкие снаряды** — стрела и файрбол теперь поддерживают спрайт-анимацию (fly/hit), вращение по направлению полёта. Fallback — улучшенная векторная стрела и градиентный файрбол.
- 🆕 **Админ-панель** (`admin.html`) — отдельная страница для настройки игры без редактирования кода:
  - Авторизация: пинкод (4-6 цифр) + Telegram ID как основной метод
  - Вкладки: Герои / Снаряды / Звуки / Названия / Пинкод
  - Настройка спрайтов: файл, число кадров, FPS, строка в листе, звук, вращение
  - Переименование классов героев, всех NPC и зданий (иконка + название + подпись)
  - Кнопки «Сохранить» и «Сбросить к умолчаниям»
- 🆕 **NAMES_CONFIG** — система переименований для героев/NPC/зданий, хранится в `localStorage`, применяется в игре мгновенно.
- 🆕 Дома на карте деревни перекомпонованы: три дома 4×4 в верхней части карты, двери в нижнем левом углу каждого дома.

### v5.0.4 (предыдущая)
**Исправления:**
- 🐛 Игра ускорялась/тормозила в зависимости от частоты обновления экрана (60 / 120 / 144 Hz) — вся логика была завязана на количество кадров `requestAnimationFrame`, а не на реальное время. Введён `delta time (dt)`, нормализованный к 60 FPS: скорость движения, кулдауны атак, таймеры частиц, текста урона, снарядов, стана врагов — теперь идут одинаково на любом устройстве независимо от FPS.
- 🐛 На iPhone (iOS Safari) нельзя было одновременно двигаться джойстиком и атаковать/пить зелье — срабатывало только одно касание. Кнопки атаки, магии, контекстного действия и слотов инвентаря переведены с `click` на `touchstart`, что даёт настоящую поддержку мультитача (джойстик одним пальцем + кнопка другим — работают параллельно).

**Изменения:**
- 🔧 Убрана надпись «Играй без авторизации» в главном меню — теперь при отсутствии Telegram-пользователя поле просто пустое.

## 🚀 Установка

Разархивируй в папку и открой `index.html` в браузере или задеплой на GitHub Pages / Telegram Mini App.

## 🖼️ Своё изображение на сплэш

Положи файл `assets/splash.png` (рекомендуется 360×360 px) — он автоматически появится на загрузочном экране.
