# Спрайты героев

## Спецификация листа (warrior.png / archer.png / mage.png)

| Параметр | Значение |
|---|---|
| Размер кадра | 48×48 px |
| Кадров в строке | 4 |
| Строк | 5 |
| Итого размер листа | 192×240 px |
| Направление | Только LEFT (вправо — зеркалится автоматически) |
| Прозрачность | PNG с альфа-каналом |
| Сглаживание | Выключено (pixel-art без антиалиасинга) |

## Порядок строк (сверху вниз)

| Строка | Анимация | Кадров |
|---|---|---|
| 0 | idle | 4 |
| 1 | walk | 4 |
| 2 | attack | 4 |
| 3 | ability (навык) | 4 |
| 4 | death | 4 |

## Порядок кадров (слева направо)

Кадры идут последовательно 0→1→2→3, затем зацикливаются (кроме death — останавливается на кадре 3).

## Имена файлов

- `warrior.png` — Мечник
- `archer.png` — Лучник
- `mage.png` — Маг

## Ракурс

3/4 сверху, персонаж смотрит влево.
